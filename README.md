# ChatApp
A chat application with using Java Socket (Client - Server)
Technologies : Java, Intellij, MySQL, Hibernate

## Installation
- Install mySql
- Edit persistence.xml file to connect database
- Edit pom.xml according to your repository
- Run `Server.java`
- Run `Client.java`
